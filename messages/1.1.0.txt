## 1.1.0

Release Date: 2018-01-08

- Updated regex matching pattern to support markdownlint's new output format.

  IMPORTANT: You will need to update to the latest markdownlint-cli (v0.6+) in
  order to lint markdown files.

  To update to the latest markdownlint-cli package using npm, run the following
  command from a terminal or command-prompt:

    npm install -g markdownlint-cli
